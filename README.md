# Amnesia-like
### Amnesia-like - это 2D игра, цель которой - выбраться из лабиринта, оставшись в живых. В это время игрока будут преследовать враждебно настроенные мобы.

#### Техническое задание:
##### Требуется написать игру, состоящую из нескольких уровней, с видом сверху. На уровнях должны присутствовать персонажи, предметы, генератор. Цель игры: набрать необхлдимое количество есдиниц топлива для генератора и покинуть уровень. Для этого нужно реализовать:
###### 1. Создание уровней.
###### 2. Создание персонажа. (Управление, взаимодейсвие с окружением)
###### 3. Создание предметов. (Генерируются на карте случайным образом)
###### 4. Создание мобов. (Поведение - должны приследовать игрока)
###### 5. Генератор, как точка выхода из уровня.
###### 6. Меню игры.
###### 7. Рекорды, поставленные игроками.
###### 8. Создание понятного интерфейса и удобного управления.
